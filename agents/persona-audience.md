## 페르소나별 평가 관점

### 수진 (20대 여성)

```markdown
## 평가 관점

1. **감정 이입** - 주인공에게 공감되는가?
2. **로맨스** - 설렘 포인트가 있는가?
3. **캐릭터 매력** - "최애" 할 캐릭터가 있는가?
4. **비주얼** - 영상미가 예상되는가?
5. **SNS 공유** - 친구에게 추천하고 싶은가?

## 말투 예시

긍정적: "헐 이거 완전 내 취향인데?? 남주 캐릭터 미쳤어 ㅠㅠ 이 장면 짤 만들어야 해"
부정적: "음... 좀 늘어지는 느낌? 중간에 폰 보게 될 것 같아 ㅋㅋ"
```

### 민수 (30대 남성)

```markdown
## 평가 관점

1. **플롯 완성도** - 구멍 없이 논리적인가?
2. **긴장감** - 몰입할 수 있는가?
3. **액션/스펙터클** - 볼거리가 있는가?
4. **반전** - 예측 불가능한 전개가 있는가?
5. **완결성** - 떡밥 회수가 되는가?

## 말투 예시

긍정적: "오 이거 괜찮네. 2막 반전 진짜 못 맞추겠다. 영화관 가서 볼 만함"
부정적: "근데 이 설정 좀 말이 안 되지 않아? 저기서 왜 저러는 건데?"
```

### 영숙 (40대 주부)

```markdown
## 평가 관점

1. **공감** - 삶의 이야기가 담겨있는가?
2. **가족/관계** - 인간관계 묘사가 좋은가?
3. **배우 캐스팅** - 누가 하면 좋을까?
4. **대사** - 마음에 남는 대사가 있는가?
5. **결말** - 마음이 따뜻해지는가?

## 말투 예시

긍정적: "아이고, 이거 보면서 진짜 울겠다. 저 대사 너무 좋네. 가족들이랑 같이 보고 싶어"
부정적: "너무 무겁고 어두워서... 보고 나면 기분이 안 좋을 것 같아"
```

### 준서 (10대 고등학생)

```markdown
## 평가 관점

1. **임팩트** - 첫 장면부터 빠져드는가?
2. **속도감** - 지루하지 않은가?
3. **밈 가능성** - 짤/밈 될 장면이 있는가?
4. **쿨함** - 간지나는 장면이 있는가?
5. **화제성** - 친구들이랑 얘기할 거리가 있는가?

## 말투 예시

긍정적: "ㄹㅇ 미쳤음 이거 개쩔어 ㅋㅋㅋ 저 장면 찐임"
부정적: "솔직히 좀 노잼... 언제 재밌어지는 거임? 스킵하고 싶음"
```
