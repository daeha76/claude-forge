# 플랫폼별 스펙

## TikTok

### 슬라이드쇼 (Photo Mode)

| 항목 | 스펙 |
|------|------|
| 최소 이미지 | 2장 |
| 최대 이미지 | 35장 |
| 권장 이미지 | 6-10장 |
| 비율 | 9:16 (필수) |
| 해상도 | 1080x1920px 이상 |
| 파일 형식 | JPEG, PNG |
| 파일 크기 | 이미지당 최대 20MB |
| 캡션 | 최대 2,200자 |
| 해시태그 | 최대 ~100개 (3-5개 권장) |

### TikTok 게시 규칙

- **트렌딩 사운드 필수**: 슬라이드쇼에 사운드가 있으면 알고리즘 가산점
- **초안 게시 권장**: Postiz로 SELF_ONLY 게시 → 앱에서 사운드 추가 → 공개
- **매일 1-3개**: 하루 3개 이하 권장 (스팸 판정 방지)
- **계정 워밍업**: 신규 계정은 7-14일 시청만 후 게시 시작

### TikTok 최적 게시 시간 (한국, KST)

| 시간 | 타겟 | 이유 |
|------|------|------|
| 07:00-08:30 | 출근족 | 출근 중 스크롤 |
| 12:00-13:00 | 직장인 | 점심시간 |
| 16:00-17:00 | 학생/직장인 | 오후 휴식 |
| 19:00-21:00 | 전체 | 저녁 황금시간 |
| 21:00-23:00 | MZ세대 | 취침 전 스크롤 |

**권장 스케줄**: 07:30, 16:30, 21:00

### TikTok 알고리즘 팁

1. **첫 1시간이 핵심**: 게시 후 1시간 내 참여도가 도달률 결정
2. **완주율 > 좋아요**: 슬라이드 전체를 넘긴 비율이 가장 중요
3. **공유 가산점**: 공유 1회 = 좋아요 10회 가치
4. **댓글 유도**: 댓글이 많으면 FYP 노출 증가
5. **해시태그 전략**: 대형(100M+) 1개 + 중형(1M-10M) 2개 + 소형(10K-100K) 2개

## Instagram

### 캐러셀 (Carousel Post)

| 항목 | 스펙 |
|------|------|
| 최소 이미지 | 2장 |
| 최대 이미지 | 10장 (20장으로 확대 중) |
| 권장 이미지 | 6-10장 |
| 비율 | 1:1, 4:5, 9:16 (모두 가능) |
| 권장 비율 | 4:5 (세로, 피드 최대 면적) |
| 해상도 | 1080x1350px (4:5) or 1080x1920px (9:16) |
| 파일 형식 | JPEG, PNG |
| 파일 크기 | 이미지당 최대 8MB |
| 캡션 | 최대 2,200자 |
| 해시태그 | 최대 30개 (5-10개 권장) |

### Instagram 게시 규칙

- **캐러셀 선호**: Instagram 알고리즘이 캐러셀을 더 밀어줌
- **첫 장이 피드 표지**: 가장 매력적인 이미지를 첫 장에
- **9:16 슬라이드**: Reels 탭에도 노출될 가능성 있음
- **자동 게시 가능**: Postiz를 통해 API로 직접 게시 (초안 불필요)
- **캐러셀 재노출**: 사용자가 첫 장만 보면, 다음에 두 번째 장으로 다시 노출

### Instagram 최적 게시 시간 (한국, KST)

| 시간 | 타겟 | 이유 |
|------|------|------|
| 07:00-09:00 | 전체 | 아침 루틴 |
| 11:30-13:00 | 직장인 | 점심시간 |
| 17:00-19:00 | 전체 | 퇴근 후 |
| 20:00-22:00 | MZ세대 | 저녁 황금시간 |

### Instagram 알고리즘 팁

1. **저장이 핵심**: 저장 수 = 가장 강력한 시그널
2. **캐러셀 완주율**: 모든 장을 넘긴 비율이 중요
3. **Reels 연계**: 같은 콘텐츠를 Reels로도 제작
4. **스토리 연동**: 게시 후 스토리에 공유하여 도달 확대
5. **Alt Text**: 접근성 + SEO, 각 이미지에 alt text 추가 권장

## 크로스 플랫폼 전략

### 동일 콘텐츠 적응

| 요소 | TikTok | Instagram |
|------|--------|-----------|
| 비율 | 9:16 | 9:16 (or 4:5) |
| 사운드 | 트렌딩 필수 | 선택 |
| CTA | "Link in bio" | "저장해두세요" |
| 해시태그 | 3-5개 | 5-10개 |
| 캡션 | 짧게 (50-100자) | 중간 (100-300자) |

### 게시 순서 (권장)

```
1. TikTok (초안) → 사운드 추가 → 게시  ← 가장 바이럴 가능성 높음
2. Instagram (캐러셀) → 자동 게시        ← 저장/캐러셀 알고리즘
```

### 플랫폼별 조정 사항

**TikTok → Instagram 변환 시:**
- 캡션 길이 늘리기 (스토리텔링 추가)
- 해시태그 추가 (10개까지)
- CTA 변경 ("Save this" / "저장해두세요")

## 이미지 사이즈 요약

| 용도 | 비율 | 해상도 | 파일 크기 |
|------|------|--------|----------|
| TikTok 슬라이드 | 9:16 | 1080x1920 | <20MB |
| Instagram 캐러셀 | 4:5 or 9:16 | 1080x1350 or 1080x1920 | <8MB |
| **우리 기본값** | **9:16** | **2K (Gemini)** | 자동 |

**결론**: 9:16 비율 2K 해상도로 생성하면 2개 플랫폼 모두 커버 가능.

## 안전 영역 (Safe Zone)

모든 플랫폼에서 UI 요소를 피해야 하는 영역:

```
┌────────────────────┐
│ ████████████████ │ ← 상단 10%: 시스템 바, 뒤로가기
│                    │
│  ┌──────────────┐  │
│  │              │  │
│  │   안전 영역   │  │ ← 15-75%: 메인 콘텐츠
│  │              │  │
│  └──────────────┘  │
│                    │
│              ████  │ ← 우측: 좋아요/댓글/공유 버튼
│ ████████████████ │ ← 하단 25%: 캡션, 음악, 네비게이션
└────────────────────┘
```

**텍스트 오버레이 안전 영역: 상단 20% ~ 40%**
