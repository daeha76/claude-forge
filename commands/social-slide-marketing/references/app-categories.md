```
├─ 개인 소비자 대상 (B2C)

│  ├─ 미용/패션 → Beauty
│  ├─ 음식/요리 → Food
│  ├─ 홈/라이프 → Home
│  ├─ 공부/학습 → Education
│  ├─ 쇼핑/구매 → E-commerce
│  ├─ 외식/카페 → F&B
│  └─ 돈/투자 → Finance
├─ 비즈니스 대상 (B2B)
│  └─ 업무 도구 → SaaS
└─ 범용
   └─ 시간/효율 → Productivity
```
