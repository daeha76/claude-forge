## Template 1: 트렌드 분석 리포트

**용도**: 경제 지표의 시계열 분석
**예시**: "한국 GDP 성장률 10년 추이", "미국 CPI 변화 분석"

### 구조

```markdown
# [지표명] 트렌드 분석 리포트

> 분석 기간: YYYY년 ~ YYYY년
> 작성일: YYYY-MM-DD
> 데이터 출처: [출처명]

---

- 가장 주목할 변곡점
- 현재 수준과 의미
```

### 작성 가이드

- Executive Summary는 바쁜 의사결정자가 이것만 읽어도 핵심을 파악할 수 있도록 작성
- 기간별 분석에서는 구조적 변곡점(정책 변화, 위기 등)을 기준으로 구분
- 차트는 최소 2개: 전체 추이 + 성장률/변화율
- 인사이트는 "X는 Y 기간 동안 Z% 증가하여 역대 최고치를 기록" 형태로 구체적 수치 포함

---

## 공통 작성 원칙

1. **데이터 우선**: 모든 주장은 수치로 뒷받침
2. **출처 명시**: 데이터 소스와 기준일 기재
3. **차트 필수**: 최소 2~3개 시각화 포함
4. **인사이트 구체화**: "증가했다" 대신 "X% 증가하여 Y를 기록"
5. **한계 투명**: 데이터 제한사항과 가정 명시
6. **액션 지향**: 결론에서 시사점과 다음 단계 제시

## 인사이트 문장 패턴

```
# 트렌드 패턴
- [X]는 YYYY년부터 YYYY년까지 [기간] 동안 연평균 [Z]% 성장했다.
- [X]는 YYYY년 [값]으로 역대 최고치/최저치를 기록했다.
- [X]는 YYYY년 [이벤트] 이후 [방향]세를 보이고 있다.

# 비교 패턴
- 한국의 [X]는 OECD 평균([값]) 대비 [Y]% 높아/낮아 [N]위를 기록했다.
- [국가A]는 [X] 부문에서 [국가B] 대비 [Y]배의 격차를 보인다.
- [X] 기준 상위 3개국은 [A], [B], [C]이며, 한국은 [N]위이다.

# 사업 근거 패턴
- [X] 시장은 YYYY년 [규모]에서 YYYY년 [규모]로 CAGR [Y]% 성장이 예상된다.
- [타겟층]의 [행동/소비] 비율은 YYYY년 [X]%에서 YYYY년 [Y]%로 증가했다.
- 상위 [N]개 기업의 시장 점유율은 [X]%로, [진입 가능성] 여지가 있다.
```
